## input 输入框

用于页面中展示重要的提示信息。

### 基本用法

页面中的非浮层元素，不会自动消失。

:::demo 手游ui。
```html
<template>
  <sy-input
    title="成功提示的文案"
    type="success">
  </sy-input>
  <sy-input
    title="消息提示的文案"
    type="info">
  </sy-input>
  <sy-input
    title="警告提示的文案"
    type="warning">
  </sy-input>
  <sy-input
    title="错误提示的文案"
    type="error">
  </sy-input>
</template>
```
:::
